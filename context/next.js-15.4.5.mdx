# Documentación Detallada de Next.js v15.4.5

Este archivo sirve como un mapa completo de la documentación de Next.js, organizada en dos secciones principales: App Router y Arquitectura.

---

- **03-layouts-and-pages.mdx**: Uso de Layouts y Pages.

- **06-partial-prerendering.mdx**: Renderizado parcial pre-renderizado.
- **07-fetching-data.mdx**: Estrategias para la obtención de datos.
- **08-updating-data.mdx**: Mutaciones y actualización de datos.
- **09-caching-and-revalidating.mdx**: Caching y revalidación de datos.
- **10-error-handling.mdx**: Manejo de errores.

- **12-images.mdx**: Optimización de imágenes con `next/image`.

### 1.3. Referencia de API (`03-api-reference/`)
Documentación técnica detallada de las APIs de Next.js.
- **01-directives/**: Directivas como `use client` y `use server`.
- **02-components/**: API de componentes como `<Image>`, `<Link>`, etc.
- **03-file-conventions/**: Convenciones de nombrado de archivos (`page.js`, `layout.js`, `error.js`, etc.).
    - **01-metadata/**: Archivos para metadatos como `sitemap.xml` y `robots.txt`.
- **04-functions/**: API de funciones como `cookies`, `headers`, `redirect`, `notFound`, etc.
- **05-config/**: Opciones de configuración en `next.config.js`.
- **06-cli/**: Referencia de la línea de comandos (`create-next-app`, `next dev`, etc.).
- **07-edge.mdx**: APIs para Edge Runtime.
- **08-turbopack.mdx**: Información sobre Turbopack.
