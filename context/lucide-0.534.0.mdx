# Documentación Detallada de Lucide v0.534.0

Este archivo sirve como un mapa completo de la documentación de la librería de iconos Lucide. A continuación se detallan todas las secciones y archivos disponibles.

## Archivos Principales (Raíz)
Documentación general y de inicio rápido.
- **comparison.md**: Comparativa entre Lucide y otras librerías de iconos populares.
- **how-to-use-icons.md**: Guía básica sobre cómo implementar y usar los iconos.
- **index.md**: Página de introducción principal a la documentación de Lucide.
- **installation.md**: Instrucciones detalladas para instalar Lucide en varios entornos.

## Conceptos Básicos (`basics/`)
Guías fundamentales para el uso diario de los iconos.
- **color.md**: Cómo aplicar colores a los iconos.
- **sizing.md**: Cómo cambiar el tamaño de los iconos.
- **stroke-width.md**: Cómo ajustar el grosor del trazo de los iconos.
- **examples/**: Contiene ejemplos de código para los conceptos básicos.
    - **styles.css**: Hoja de estilos para los ejemplos.
    - **absolute-stroke-width-icon/**: Ejemplo de grosor de trazo absoluto.
    - **button-example/**: Ejemplo de uso de iconos en botones.
    - **color-icon/**: Ejemplo de aplicación de color.
    - **size-icon-css-example/**: Ejemplo de cambio de tamaño con CSS.
    - **size-icon-example/**: Ejemplo básico de cambio de tamaño.
    - **size-icon-font-example/**: Ejemplo de cambio de tamaño basado en la fuente.
    - **size-icon-tailwind-example/**: Ejemplo de cambio de tamaño con Tailwind CSS.
    - **stroke-width-icon/**: Ejemplo de ajuste del grosor del trazo.

## Uso Avanzado (`advanced/`)
Técnicas y personalizaciones más complejas.
- **accessibility.md**: Guía para asegurar la accesibilidad de los iconos.
- **aliased-names.md**: Cómo usar alias para los nombres de los iconos.
- **filled-icons.md**: Instrucciones para usar o crear iconos con relleno.
- **global-styling.md**: Cómo aplicar estilos globales a todos los iconos.
- **examples/**: Contiene ejemplos de código para los conceptos avanzados.
    - **filled-icon-example/**: Ejemplo de un icono con relleno.
    - **global-styling-absolute-strokewidth-example/**: Ejemplo de estilo global con grosor de trazo absoluto.
    - **global-styling-css-example/**: Ejemplo de estilo global usando CSS.

## Diseño (`design/`)
Guías para diseñadores que trabajan con o contribuyen a Lucide.
- **affinity-designer-guide.md**: Guía para usar los iconos en Affinity Designer.
- **figma-guide.md**: Guía para usar el kit de UI de Lucide en Figma.
- **icon-design-guide.md**: Principios y guías para diseñar iconos para Lucide.
- **illustrator-guide.md**: Guía para usar los iconos en Adobe Illustrator.
- **inkscape-guide.md**: Guía para usar los iconos en Inkscape.

## Paquetes (`packages/`)
Documentación específica para cada paquete de integración de Lucide.
- **lucide-angular.md**: Para integración con Angular.
- **lucide-astro.md**: Para integración con Astro.
- **lucide-preact.md**: Para integración con Preact.
- **lucide-react-native.md**: Para integración con React Native.
- **lucide-react.md**: Para integración con React.
- **lucide-solid.md**: Para integración con Solid.js.
- **lucide-static.md**: Para usar los iconos como archivos estáticos.
- **lucide-svelte.md**: Para integración con Svelte.
- **lucide-vue-next.md**: Para integración con Vue 3.
- **lucide-vue.md**: Para integración con Vue 2.
- **lucide.md**: Documentación del paquete principal de Lucide.
