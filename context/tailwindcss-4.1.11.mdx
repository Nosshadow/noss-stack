# Documentación Detallada de Tailwind CSS v4.1.11

Este archivo sirve como un mapa completo de la documentación de Tailwind CSS, organizada por categorías para una fácil referencia.

## Conceptos Fundamentales (`core-concepts/`)
- **adding-custom-styles.mdx**: Cómo añadir y extender estilos personalizados.
- **compatibility.mdx**: Información sobre la compatibilidad con diferentes herramientas y frameworks.
- **dark-mode.mdx**: Guía para implementar el modo oscuro.
- **detecting-classes-in-source-files.mdx**: Cómo Tailwind detecta las clases en tus archivos.
- **editor-setup.mdx**: Configuración recomendada para editores de código.
- **functions-and-directives.mdx**: Uso de las funciones y directivas de Tailwind en tu CSS.
- **hover-focus-and-other-states.mdx**: Cómo aplicar estilos a diferentes estados (hover, focus, etc.).
- **preflight.mdx**: Sobre el reseteo de estilos base que aplica Tailwind.
- **responsive-design.mdx**: Diseño responsivo con las utilidades de Tailwind.
- **styling-with-utility-classes.mdx**: La filosofía de estilizar con clases de utilidad.
- **theme.mdx**: Cómo personalizar el tema de Tailwind.
- **upgrade-guide.mdx**: Guía para actualizar a nuevas versiones.

## Layout (`layout/`)
- **box-sizing.mdx**: Controla cómo se calcula el tamaño total de un elemento.
- **break-after.mdx, break-before.mdx, break-inside.mdx**: Controlan cómo se rompen las columnas o páginas.
- **clear.mdx**: Controla el ajuste de elementos alrededor de elementos flotantes.
- **columns.mdx**: Define el número de columnas en un layout de múltiples columnas.
- **display.mdx**: Define el comportamiento de la caja de un elemento (block, inline, flex, etc.).
- **float.mdx**: Posiciona un elemento a la izquierda o derecha de su contenedor.
- **isolation.mdx**: Controla si un elemento debe crear un nuevo contexto de apilamiento.
- **object-fit.mdx**: Define cómo el contenido de un `<img>` o `<video>` se ajusta a su contenedor.
- **object-position.mdx**: Especifica la alineación del contenido de un elemento.
- **overflow.mdx**: Controla cómo se muestra el contenido que se desborda de un elemento.
- **position.mdx**: Define el método de posicionamiento de un elemento (static, relative, absolute, etc.).
- **top-right-bottom-left.mdx**: Controla la posición de elementos posicionados.
- **visibility.mdx**: Muestra u oculta un elemento.
- **z-index.mdx**: Controla el orden de apilamiento de elementos posicionados.

## Flexbox & Grid (`flexbox-grid/`)
- **flex.mdx, flex-basis.mdx, flex-direction.mdx, flex-grow.mdx, flex-shrink.mdx, flex-wrap.mdx**: Propiedades para layouts con Flexbox.
- **gap.mdx**: Controla el espacio entre elementos de grid y flexbox.
- **grid-auto-columns.mdx, grid-auto-flow.mdx, grid-auto-rows.mdx**: Controlan el flujo y las columnas/filas automáticas en CSS Grid.
- **grid-column.mdx, grid-row.mdx**: Controlan la ubicación de un elemento en la cuadrícula.
- **grid-template-columns.mdx, grid-template-rows.mdx**: Definen las columnas y filas de la cuadrícula.
- **justify-content.mdx, justify-items.mdx, justify-self.mdx**: Propiedades de alineación para Flexbox y Grid.
- **align-content.mdx, align-items.mdx, align-self.mdx**: Propiedades de alineación para Flexbox y Grid.
- **place-content.mdx, place-items.mdx, place-self.mdx**: Propiedades de alineación para Flexbox y Grid.
- **order.mdx**: Controla el orden de los elementos en un contenedor flex.

## Espaciado (`spacing/`)
- **margin.mdx**: Controla el margen de un elemento.
- **padding.mdx**: Controla el padding de un elemento.

## Tamaño (`sizing/`)
- **width.mdx, min-width.mdx, max-width.mdx**: Controlan el ancho de un elemento.
- **height.mdx, min-height.mdx, max-height.mdx**: Controlan la altura de un elemento.
- **aspect-ratio.mdx**: Controla la relación de aspecto de un elemento.
- **field-sizing.mdx**: Controla el tamaño de los campos de formulario.

## Tipografía (`typography/`)
- **font-family.mdx, font-size.mdx, font-smoothing.mdx, font-style.mdx, font-weight.mdx, font-variant-numeric.mdx, font-stretch.mdx**: Propiedades para controlar la fuente.
- **letter-spacing.mdx, line-clamp.mdx, line-height.mdx**: Controlan el espaciado y la altura de línea del texto.
- **list-style-image.mdx, list-style-position.mdx, list-style-type.mdx**: Propiedades para listas.
- **text-align.mdx, color.mdx, colors.mdx**: Alineación y color del texto.
- **text-decoration-color.mdx, text-decoration-line.mdx, text-decoration-style.mdx, text-decoration-thickness.mdx, text-underline-offset.mdx**: Propiedades para la decoración del texto.
- **text-transform.mdx, text-overflow.mdx, text-indent.mdx, text-shadow.mdx, text-wrap.mdx**: Otras propiedades de texto.
- **vertical-align.mdx, white-space.mdx, word-break.mdx, overflow-wrap.mdx**: Controlan la alineación vertical y el salto de línea.
- **content.mdx**: Para usar con los pseudo-elementos `::before` y `::after`.
- **hyphens.mdx**: Controla la separación de palabras con guiones.

## Fondos (`backgrounds/`)
- **background-attachment.mdx, background-clip.mdx, background-color.mdx, background-origin.mdx, background-position.mdx, background-repeat.mdx, background-size.mdx, background-image.mdx, background-blend-mode.mdx**: Propiedades para estilizar fondos.

## Bordes (`borders/`)
- **border-radius.mdx, border-width.mdx, border-color.mdx, border-style.mdx**: Propiedades para estilizar bordes.
- **border-collapse.mdx, border-spacing.mdx**: Propiedades para tablas.
- **outline-color.mdx, outline-offset.mdx, outline-style.mdx, outline-width.mdx**: Propiedades para el contorno de los elementos.

## Efectos (`effects/`)
- **box-shadow.mdx**: Añade sombras a los elementos.
- **opacity.mdx**: Controla la opacidad de un elemento.
- **mix-blend-mode.mdx**: Controla cómo se mezcla un elemento con su fondo.
- **box-decoration-break.mdx**: Controla cómo se aplican los fragmentos de un elemento.
- **mask-clip.mdx, mask-composite.mdx, mask-image.mdx, mask-mode.mdx, mask-origin.mdx, mask-position.mdx, mask-repeat.mdx, mask-size.mdx, mask-type.mdx**: Propiedades para enmascarar elementos.

## Filtros (`filters/`)
- **filter.mdx, filter-blur.mdx, filter-brightness.mdx, filter-contrast.mdx, filter-drop-shadow.mdx, filter-grayscale.mdx, filter-hue-rotate.mdx, filter-invert.mdx, filter-saturate.mdx, filter-sepia.mdx**: Filtros de CSS.
- **backdrop-filter.mdx, backdrop-filter-blur.mdx, etc.**: Filtros aplicados al fondo de un elemento.

## Tablas (`tables/`)
- **caption-side.mdx**: Controla la posición del caption de una tabla.
- **table-layout.mdx**: Controla el algoritmo de layout de una tabla.

## Transiciones y Animaciones (`transitions-animation/`)
- **transition-property.mdx, transition-duration.mdx, transition-timing-function.mdx, transition-delay.mdx, transition-behavior.mdx**: Propiedades para transiciones.
- **animation.mdx**: Propiedades para animaciones CSS.
- **will-change.mdx**: Optimiza animaciones.

## Transformaciones (`transforms/`)
- **transform.mdx, transform-origin.mdx, transform-style.mdx**: Propiedades de transformación.
- **scale.mdx, rotate.mdx, translate.mdx, skew.mdx**: Funciones de transformación.
- **perspective.mdx, perspective-origin.mdx**: Perspectiva en transformaciones 3D.
- **backface-visibility.mdx**: Visibilidad de la cara trasera de un elemento.

## Interactividad (`interactivity/`)
- **accent-color.mdx, caret-color.mdx, color-scheme.mdx**: Colores de acento y de cursor.
- **appearance.mdx**: Controla la apariencia nativa de los elementos.
- **cursor.mdx**: Cambia el cursor del ratón.
- **overscroll-behavior.mdx**: Controla el comportamiento del scroll en los límites.
- **pointer-events.mdx**: Controla si un elemento puede ser objetivo de eventos de puntero.
- **resize.mdx**: Controla si un elemento es redimensionable.
- **scroll-behavior.mdx, scroll-margin.mdx, scroll-padding.mdx, scroll-snap-align.mdx, scroll-snap-stop.mdx, scroll-snap-type.mdx**: Propiedades para el control del scroll.
- **touch-action.mdx**: Controla la respuesta a la interacción táctil.
- **user-select.mdx**: Controla si el texto puede ser seleccionado.

## SVG (`svg/`)
- **fill.mdx**: Define el color de relleno de un SVG.
- **stroke.mdx, stroke-width.mdx**: Define el color y el grosor del trazo de un SVG.

## Accesibilidad (`accessibility/`)
- **forced-color-adjust.mdx**: Controla la participación de un elemento en el modo de colores forzados.
